# Gradle Offline Access for Study

Данный репозиторий предназначен для **офлайн-работы с Android-проектами** в условиях ограничений
по дисциплине **МДК.01.03 Разработка мобильных приложений**.

Gradle-кеш подготовлен заранее, чтобы:
- не зависеть от интернета;
- избежать проблем с доступом к Google и Maven.

---

## Что требуется перед началом работы

На компьютере должно быть установлено:
- **Android Studio** (рекомендуется версия не ниже 2024);
- **JDK 17 или выше** (или встроенный JBR из Android Studio);
- ОС Windows 10 / 11.

Интернет **не требуется**, если выполнены шаги ниже.
